# Stride Lab

O objetivo principal desse projeto é a busca de aprendizado e aperfeiçoamento em desenvolvimento, onde vamos desenvolver uma plataforma pensando em ajudar a operação de assessorias esportivas, personal trainer e outros profissionais da área de educação física e esportes.

Vamos trabalhar com Laravel e Nuxt como stack de desenvolvimento, aplicando o uso de GitHub Actions para fazer deploy, usar um ambiente de desenvolvimento com docker, trabalhar com testes unitários e buscando desenvolver um projeto bem estruturado e o mais próximo possível de um produto bem profissional (na medida do possível e do nosso conhecimento).
